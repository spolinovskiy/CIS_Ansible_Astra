# Required Information

Schema: `canonical-required-information-v1`

This directory contains placeholders for information required before safe audit or remediation in an offline OT environment. Do not store production secrets here.

The template currently defines 89 classified items across 10 groups. Each item has `requirement_level`, `placeholder`, `why_needed`, `sensitive`, and `example_allowed`.

`example_lab_values.yml` is for lab examples only and must not be used as production inventory data.
